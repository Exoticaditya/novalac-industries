## ğŸ”§ Troubleshooting

### Image Not Showing?
1. **Check filename** - Must match exactly (case-sensitive)
2. **Check location** - Must be in correct folder
3. **Check format** - Must be JPG or PNG
4. **Clear cache** - Press Ctrl+Shift+R to hard refresh
5. **Check file size** - Should be under 2MB

### Image Looks Distorted?
1. **Check dimensions** - Should be at least 800x800 for products
2. **Maintain aspect ratio** - Don't stretch images
3. **Use proper format** - JPG for photos, PNG for graphics

### Image Too Large?
1. **Compress** using online tools
2. **Resize** to recommended dimensions
3. **Optimize** for web using image editors
